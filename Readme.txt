I. Required library to use the document:
    +) Numpy
    +) Matplotlib
    +) Scipy
    +) Opencv
    +) PIL
    +) tqdm

    ==> load the library via pip:
        +) pip install numpy
        +) pip install matplotlib
        +) pip install scipy
        +) pip install opencv-python
        +) pip install Pillow
        +) pip install tqdm
